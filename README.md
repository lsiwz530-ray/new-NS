# NorthSite

متجر مفاتيح رقمية مع لوحة تحكم كاملة. React + Vite (فرونت) و Express + SQLite (باكند).

## التشغيل محليًا

```bash
npm install
npm run dev
```

- الواجهة: http://localhost:5173
- الـ API: http://localhost:3001 (بروكسي تلقائي من الفرونت)
- لوحة التحكم: `/auth`  المستخدم: `North`  الباسورد: `North123`

## النشر على Railway

1. أنشئ خدمة Node جديدة.
2. اضبط أوامر التشغيل:
   - Build: `npm install && npm run build`
   - Start: `npm run start`
3. عيّن متغير `PORT` (Railway يعينه تلقائيًا).
4. البيانات تُحفظ في `data/northsite.db` — أضف Volume على المسار `/app/data` عبر Railway لضمان بقاء البيانات بعد إعادة النشر (`DATA_DIR=/app/data`).

> ملاحظة: `npm run start` يشغّل السيرفر فقط، وهو يقدّم ملفات `dist/` الناتجة من `npm run build`.
